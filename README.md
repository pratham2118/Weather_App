# 🌦️ Weather App

A GUI-based Weather Application built using Python.

## 🚀 Features
- Real-time weather data
- City-based search
- Temperature, humidity, wind details
- Clean GUI interface

## 🛠️ Tech Stack
- Python
- Tkinter
- Weather API

## ▶️ How to Run

```bash
pip install -r requirements.txt
python weather_app/weather_gui.py
